# ghostty-config

My personal [Ghostty](https://ghostty.org) terminal configuration for macOS.

## Usage

Copy the contents of `config` into your Ghostty config file, then reload Ghostty with `Cmd+Shift+,`.

## Highlights

- Translucent background with blur effect
- Window state always saved and restored
- Command finish notifications
- Large scrollback buffer (25M lines)
- Selection auto-copied to clipboard
- Clipboard paste protection
- Zsh shell integration

### Keybindings

| Shortcut | Action |
|---|---|
| `Cmd+D` | Split pane to the right |
| `Opt+Space` (global) | Toggle Ghostty visibility |
